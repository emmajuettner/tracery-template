This is an example of how you can put a Tracery project on a website, for people to interact with. I made this template project because I wanted a simple starting point for displaying Tracery projects. Maybe you will find it useful too! You can review the code in the github repo.

A caveat: I know only enough javascript to slap things together with duct tape, so it's very probable that there are better ways to put this logic together. Feel free to suggest improvements or use this as a jumping-off point to implement something better!

This example uses a very simple set of tracery grammars that generate a description of a dragon. Click the button below to generate a dragon.
